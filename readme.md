# Monitor de Estado de Servicios

Este proyecto proporciona una forma visual y práctica de supervisar sitios y servicios web.

- Qué hace: muestra el estado (funciona / error) y la latencia (ms) de cada sitio, guarda un historial corto y alerta cuando hay fallos o latencias críticas.
- Para quién: equipos de operaciones ligeros, responsables de infraestructura y desarrolladores que quieren detectar problemas antes de que los usuarios los noten.

Cómo usar (resumen, sin detalles técnicos):

- Abrir `index.html` en un navegador compatible.
- El tablero actualiza automáticamente cada pocos minutos y guarda un historial temporal en el navegador.
- Puedes alternar temas visuales y ver una leyenda con los umbrales de latencia en la página de `leyenda.html`.

Detalles técnicos, configuración y guía de despliegue están en la carpeta `docs/` (ahí están las instrucciones más técnicas y archivos de referencia).
